
# Консольная часть

## Доступные команды

### Управление задачей
- `add <Название>` - Добавляет новую задачу
- `list` - Выводит все задачи
- `done <id>` - Отмечает задачу выполненной
- `delete <id>` - Удаляет задачу

### Опции для list
- `list --pending` - Выводит все активные
- `list --complited` - Выводит все завершенные
